# Bibliographie

Petit projet "exercice" dans le cadre d'un apprentissage du langage **RUST**.

## Ressources

### Général

- [Rust Book](https://doc.rust-lang.org/stable/book/) - Chapitres 1 à 8
- Les [constantes](https://doc.rust-lang.org/rust-by-example/custom_types/constants.html)
- Les [Vecteurs](https://doc.rust-lang.org/std/vec/struct.Vec.html)
- les [itérateurs](https://doc.rust-lang.org/std/iter/trait.Iterator.html)

### Crates

- [chrono](https://docs.rs/chrono/latest/chrono/)
- **lazy-static**:
  - [Documentation crate](https://docs.rs/lazy_static/latest/lazy_static/)
  - [Démystifier le modèle lazy_static de Rust](https://blog.logrocket.com/rust-lazy-static-pattern/)
- [heck](https://docs.rs/heck/latest/heck/)
- [std](https://doc.rust-lang.org/std/)

### Méthodes

- [Suppression de préfixes](https://doc.rust-lang.org/std/primitive.str.html#method.trim_start_matches) dans une chaîne de caractères
- [Changer un caractère](https://doc.rust-lang.org/std/string/struct.String.html#method.replace) en autre caractère
- [Filtrage](https://doc.rust-lang.org/std/iter/trait.Iterator.html#method.filter) sur une itération
- [Tri](https://doc.rust-lang.org/std/primitive.slice.html#method.sort_by_key) à l'aide d'une clé
